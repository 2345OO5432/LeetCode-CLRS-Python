###412. Fizz Buzz

题目:
<https://leetcode.com/problems/fizz-buzz/>


难度:
Easy


```
class Solution(object):
    def fizzBuzz(self, n):
        """
        :type n: int
        :rtype: List[str]
        """
        res = []

        for i in range(1,n+1):
        	item = str(i)
        	if i % 3 == 0 and i % 5 == 0:
        		item = 'FizzBuzz'
        	elif i % 3 == 0:
        		item = 'Fizz'
        	elif i % 5 == 0:
        		item = 'Buzz'
        	res.append(item)
        return res


```
