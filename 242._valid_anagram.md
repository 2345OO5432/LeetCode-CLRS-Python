class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        sList = sorted(list(s))
        tList = sorted(list(t))
        return sList == tList

作弊神奇python大法，看了看别人的解法，用字数统计，因为只可能是26个字母

然后发现作弊大法居然更快

```

class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        if len(s) != len(t):
            return False
        
        charCnt = [0] * 26
        
        for i in range(len(s)):
            charCnt[ord(s[i]) - 97] += 1
        
        for i in range(len(t)):
            charCnt[ord(t[i]) - 97] -= 1 
            if charCnt[ord(t[i]) - 97] < 0:
                return False
        return True
```
